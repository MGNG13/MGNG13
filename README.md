# Acerca de mí

Fundador y desarrollador Full Stack, construyendo tecnología para automatizar la generación de código.

Llevo más de 3 años liderando proyectos de software con proyección internacional. Hoy dirijo mi propia empresa, con el objetivo de cambiar la forma en que se desarrolla software mediante automatización inteligente.

**Lo que hago:**

🔧 **Desarrollo** — Full Stack con Java, TypeScript, Node.js y React. Bases de datos SQL/NoSQL, servidores Linux y ciberseguridad aplicada.

🎬 **Diseño** — Motion design 2D/3D (Cinema 4D, After Effects) desde hace más de 7 años, con trabajo para estudios como Giggle Block y animaciones para Mojang Studios (Minecraft).

🚀 **Visión** — Conectar ingeniería y creatividad para resolver problemas reales, con foco en resultados.
